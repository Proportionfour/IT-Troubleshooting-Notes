# IT Troubleshooting Notes
A running log of real-world IT issues I've encountered, diagnosed, and resolved.
Each entry documents the symptoms, what I tried, what actually worked, and why.

Intended as a personal reference and a resource for anyone facing similar problems.

---

## Issue template

Each file uses this structure:

**Title**: Short description of the problem  
**Date**: Approx. when this occurred  
**Environment**: OS, hardware, software versions, network context  
**Symptoms**: What the user/system reported  
**What I tried**: Steps taken before the fix (including dead ends)  
**Root cause**: What was actually wrong  
**Resolution**: Exact steps that fixed it  
**Notes**: Caveats, related issues, things to watch for

## Disclaimer

These are real-world notes, not official documentation. Your environment may
differ — use your judgment before applying any fix.
